# 🃏 GGPoker Action — хувь зарах самбар

Покерын тэмцээний **хувь (action / staking)** зарах вэб самбар. Хоёр хуудастай:

- **`index.html`** — нийтийн хуудас. Хувь авах хүн линкээр ороод тэмцээн, markup, үлдсэн хувь, live явц, үр дүн, ашгаа хардаг. Хувиа шууд захиалж болно.
- **`admin.html`** — зөвхөн чиний хуудас (PIN-ээр хамгаалагдсан). Тэмцээн нэмэх/хасах, markup тохируулах, live chip оруулах, үр дүн оруулах, ирсэн захиалгыг "Төлсөн" гэж баталгаажуулах.

Дата нь **Supabase** (чиний "Onosh project") дээр хадгалагдана. Бүх хүснэгт `stk_` угтвартай тул бизнесийн датанаас тусгаарлагдсан.

---

## 🔑 Хамгийн эхэнд (ЗААВАЛ)

1. `admin.html` нээ → анхны PIN: **`onosh2026`**
2. **Тохиргоо** таб → PIN-ээ заавал шинээр сольё.
3. Мөн **Тохиргоо** хэсэгт: нэр, төлбөрийн данс, холбоо барих, ханш (USD→MNT), markup-аа тааруул.
4. **Session** таб → огноо, нийт зарах хувь (%), markup, төлвөө шалга.
5. **Тэмцээн** таб → өнөөдрийн 14 тэмцээн орчихсон байгаа. Нэмэх/хасах/засах боломжтой.

> Демо болгож 2026.06.28-ны 14 тэмцээн (нийт buy-in ≈ $2,003) урьдчилж оруулсан байгаа.

## 🌐 GitHub Pages дээр байршуулах (үнэгүй)

1. [github.com](https://github.com) дээр нэвтэр → **New repository** → нэр өг (ж: `poker-action`) → **Public** → Create.
2. **Add file → Upload files** → энэ хавтасны **бүх файлыг** (`index.html`, `admin.html`, `config.js`, `styles.css`) чирч оруул → **Commit changes**.
3. Repo → **Settings → Pages** → *Source*-д **`main`** branch, `/root` сонго → **Save**.
4. 1-2 минут хүлээ. Дараах хаягууд гарна:
   - Нийтийн линк (хувь авагчдад илгээх): `https://<хэрэглэгч>.github.io/poker-action/`
   - Admin линк (зөвхөн чи): `https://<хэрэглэгч>.github.io/poker-action/admin.html`

Хувь зарахдаа **нийтийн линкээ** л Telegram/Facebook-т тавь. Admin линкээ хэнд ч бүү өг.

## 💡 Хэрхэн ажилладаг вэ (action/markup)

- **Нийт buy-in** = бүх тэмцээний buy-in нийлбэр (USD).
- **1% үнэ** = `Нийт buy-in ÷ 100 × markup`. Жишээ: $2,003 багц, markup 1.10 → 1% нь ≈ $22.04 ≈ 76,000₮.
- Хувь авагч **P%** авбал: `P% × Нийт buy-in × markup` төлнө, **P% × нийт хожил**-оо буцааж авна.
- **Markup** = чиний ашиг (edge). 1.10 = 10% нэмэгдэл. 1.00 = яг өртгөөр.
- Тэмцээн дуусаад **Үр дүн** оруулахад, хувь авагч бүрийн ашиг автоматаар тооцогдож харагдана.

## 🔁 Өдөр тутмын ажиллагаа

- **Тэмцээний өмнө:** Session-ийн markup, нийт зарах %-аа тохируул. Тэмцээнүүдээ шалга.
- **Тоглож байх үед:** Тэмцээн таб → chip count, төлөв (Амьд/Унасан), байраа шинэчил. Session төлвийг "Тоглож байна" болго. Нийтийн хуудас 20 секунд тутамд автоматаар шинэчлэгдэнэ.
- **Захиалга ирэхэд:** Захиалга таб → төлбөр ороход **Төлсөн** болго.
- **Дууссаны дараа:** Тэмцээн бүрийн **Үр дүн (USD)**-г оруул. Бүгд автоматаар тооцоологдоно.

## ➕ Дараагийн өдөр / шинэ багц

**Session** таб → **+ Шинэ session үүсгэх**. Хуучин нь идэвхгүй болж, шинэдээ тэмцээнээ нэмнэ.

## 🔒 Аюулгүй байдал

- Хувь авагч нар бусдын **утас/Telegram, төлсөн дүн (USD)**-г харж **чадахгүй** — зөвхөн нэр, хувь, төлвийг хардаг.
- Бүх засвар/үр дүн оруулах үйлдэл **admin PIN**-ээр хамгаалагдсан (сервер тал дээр шалгадаг). Тийм болохоор PIN-ээ нууцал, анхныхаа солих хэрэгтэй.
- `config.js` доторх түлхүүр бол **нийтийн (anon)** түлхүүр — ил байж болно, аюулгүй.

## 🎨 GGPoker лого солих
Одоо CSS-ээр хийсэн "GG" тэмдэг бий. Жинхэнэ лого тавихыг хүсвэл `index.html`/`admin.html` доторх `<span class="gg">...</span>` хэсгийг өөрийн зургаар сольж болно.
